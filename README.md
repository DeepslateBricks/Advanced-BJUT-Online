若您已经安装了 Tampermonkey（油猴 / 篡改猴）插件，请点击该 [**链接**](https://cdn.jsdelivr.net/gh/DeepslateBricks/Advanced-BJUT-Online/advanced-bjut-online.js) 添加此脚本。

若您还未安装上述插件，请使用该提示词询问 AI 如何安装：

